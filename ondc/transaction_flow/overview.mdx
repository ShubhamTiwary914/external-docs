# ONDC Beckn API Endpoints

This document provides an overview of the ONDC Beckn API endpoints and their classification.





<Tip>
  Initial endpoint (/search) is triggered by BAP -> BG -> BPP (Buyer to Sellers via Gateway)
</Tip>

<Tip>
  While the next APIs (/select, /init, /confirm, ..) occurs between the BAP & BPP (Buyer & Sellers Directly)
</Tip>


## Initial Endpoints

These endpoints are used to initiate requests via HTTP calls:

- `/search`
- `/select`
- `/init`
- `/confirm`


## Callback Endpoints

These endpoints handle responses to the initial requests on WebHooks:

- `/on_search`
- `/on_select`
- `/on_init`
- `/on_confirm`
