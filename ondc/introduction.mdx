# ONDC Beckn Based Backend Architecture

![Beckn](/images/ondc/becknBase.png)

The ONDC Network Backend is powered by the Beckn Protocol, which facilitates the creation of open, peer-to-peer decentralized networks for pan-sector economic transactions. This architecture can be implemented across any number of Buyer and Seller apps, forming a network of open, decentralized commerce.
